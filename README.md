## Deskripsi Proyek
Aplikasi Android berbasis Kotlin untuk memanajemen data pasien menggunakan REST API. Fitur utama dalam aplikasi ini meliputi:
* **Login Authentication (POST):** Mengirim kredensial dan menerima Bearer Token.
* **Session Management:** Menyimpan Token menggunakan `SharedPreferences`.
* **Fetch Data (GET):** Mengambil data rekam medis pasien menggunakan Bearer Token pada *Header* request.
* **UI/UX:** Menampilkan daftar pasien secara rapi menggunakan `RecyclerView` dan desain kustom.
